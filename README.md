#  Glide Test Api

## pre-requisitos
 - Python 3.5 o superior

## virtualenv

```
$ python3 -m venv venv
$ . venv/bin/activate
```
```
$ deactivate
```

## install dependencies

```
$ pip install -r requirements.txt
```

## development server

```
$ export FLASK_APP=glide_test_api
$ export FLASK_ENV=development
$ flask run
```
